

# Работа с файлами в Python

Полный учебник по работе с файлами для начинающих

## Что вы научитесь делать:
- 📝 Читать и писать текстовые файлы
- 📦 Сохранять данные в JSON
- 📊 Работать с CSV таблицами
- 🖼️ Создавать и редактировать изображения
- 🔊 Воспроизводить звуки
- 📁 Управлять папками и файлами

## Как установить зависимости:
pip install pillow pygame

## Запуск
python tutorial.py
